# List

## Inline

```
- My list: item 1, item 2, item 3
```

## Block

```
- My list
  + item 1
	+ item 2
	+ item 3
```

## Shared

```
- My book
  - Pro: item 1
	- Pro: item 2
	- Pro: item 3
```

equivalent to

```
- My book
  - Pro: item 1, item 2, item 3
```

or

```
- My book
  - Pro
	  - item 1
		- item 2
		- item 3
```
